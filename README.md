## Python Calculator Application
A versatile, command-line based calculator built with Python. This application provides a simple yet effective interface for performing essential mathematical operations, making it an ideal tool for quick calculations and understanding Python logic.
## Features
* Basic Arithmetic: Supports addition, subtraction, multiplication, and division.
* Continuous Operation: Allows users to perform multiple calculations in a single session without restarting the script.
* Error Handling: Built-in validation to handle scenarios like division by zero or invalid character inputs.
* Clean Interface: User-friendly prompts and formatted result displays.
## Technical Implementation
The project is contained in calculator.py and demonstrates key Python programming concepts:
* Functions: Modular code structure with dedicated functions for each mathematical operation.
* While Loops: Implemented to keep the application running until the user chooses to exit.
* Conditional Logic: Uses if-elif-else blocks to route user input to the correct calculation.
* Type Casting: Proper handling of string inputs and numerical conversions to ensure mathematical accuracy.
## Installation & Usage
### Prerequisites
* Python 3.x installed on your machine. You can check your version by running:
```Bash```
python --version
```Bash```
### Running the Application
1. Clone the repository:
```Bash```
git clone https://github.com/Haniatahir05/Calculator.git
```Bash```
2. Navigate to the folder:
```Bash```
cd Calculator
```Bash```
3. Run the script:
```Bash```
python calculator.py
```Bash```
## Sample Workflow
1. Select an operation from the provided menu (e.g., 1 for Add, 2 for Subtract).
2. Enter the first number.
3. Enter the second number.
4. View the calculated result.
5. Choose whether to perform another calculation or exit.
